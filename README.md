## 安装使用（Installation）

```shell
git clone https://github.com/realwds/gulp-person-website.git
npm install gulp -g
npm install
gulp release
```

node 版本应为 10+ 的稳定版本，如 10.18.1 
使用 nvm 管理 node 版本
`nvm install 10.18.1`
